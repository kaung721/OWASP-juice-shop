# OWASP Juice Shop Writeups

This repository contains my personal writeups for selected OWASP Juice Shop challenges.

## Overview
These writeups document vulnerabilities I actively solved while working through OWASP Juice Shop. I did not document every challenge—only the ones that required deeper analysis, debugging, or non-trivial exploitation paths.

Each report follows a consistent bug bounty–style format, focusing on:
- Vulnerability identification
- Reproduction steps
- Root cause analysis
- Security impact
- Suggested remediation

## Scope
Only challenges that were personally difficult or required investigation are included. Simpler or straightforward challenges are intentionally omitted.

## Reference
Official OWASP Juice Shop solutions and walkthroughs:
* https://pwning.owasp-juice.shop/part1/
* https://help.owasp-juice.shop/appendix/solutions.html

## Purpose
- Practice real-world vulnerability reporting
- Understand common web and API security flaws
- Explore prompt injection and modern LLM-related security issues
- Build familiarity with structured security documentation

## Notes
These writeups reflect my own exploitation approach and may differ from alternative solutions. If there are additional or more efficient methods, they are intentionally left open for further exploration or community comparison.
